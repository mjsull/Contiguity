    Contiguity

    Tool for visualising PacBio and Illumina assemblies, exploring assembly graphs and scaffolding.



    Requirements

    • Python 2.7+
    • NCBI-BLAST+ (needed for overlap edge creation and automatic comparison generation)
    • Bowtie 2 (needed for paired end edge creation)

    For information on how to install and use Contiguity please see Contiguity_manual.pdf available at https://github.com/mjsull/Contiguity
